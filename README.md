
## Local development

Just open `index.html` directly in your browser to preview or make changes.

## Deployment (Cloudflare Pages)

This repo is connected to **Cloudflare Pages**.

- **Production branch:** `main`
- On every push to `main`, Cloudflare:
  - Deploys the contents of the repo root to:
    - `https://[your-project].pages.dev`

**No build command is needed.**
- **Build output directory:** `.` (repo root)

## Custom domain

- `darren.my` is mapped via Cloudflare DNS to the Pages project.
